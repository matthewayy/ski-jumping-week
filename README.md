application for "ski jumping week"

after calling registration() method, user should provide name of parcitipants, if user provide nothing/just press enter,
next method, beginTournament() will be called. If user type "jump", the code of jumpingOrder() mehod will be executed, for every participant
will be randomly generated jump ( in interval 60-120 ) and judge votes. The score consist of jump + judge votes except lowest and highest vote
(for ex. if votes are [1,2,3,4,5] only 2,3,4 will be counted. If user type "exit" insted of "jump", the method tournamentResults() will be executed - prints out the positions of participants, their jumps and score.
